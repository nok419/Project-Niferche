Project Niferche デザインシステム仕様
テーマシステム概要
Project Nifercheウェブサイトは、4つの世界観に基づくテーマシステムを実装します。各世界観は独自の視覚言語を持ちながらも、一貫したUIコンポーネントと操作性を提供します。
基本方針

CSS変数を使用した一貫したテーマシステム
コンポーネントとテーマの分離による柔軟性
モバイルファーストのレスポンシブデザイン
アクセシビリティに配慮したコントラストと操作性

テーマ切り替え関数の例
typescriptfunction setTheme(theme: WorldType) {
  const root = document.documentElement;
  
  // Reset current theme
  const themes: WorldType[] = ['hodemei', 'quxe', 'alsarejia', 'laboratory'];
  themes.forEach(t => root.classList.remove(`theme-${t}`));
  
  // Set new theme
  root.classList.add(`theme-${theme}`);
  
  // Update CSS variables
  root.style.setProperty('--primary', `var(--${theme}-primary)`);
  root.style.setProperty('--secondary', `var(--${theme}-secondary)`);
  root.style.setProperty('--background', `var(--${theme}-background)`);
  root.style.setProperty('--surface', `var(--${theme}-surface)`);
  root.style.setProperty('--text', `var(--${theme}-text)`);
  root.style.setProperty('--accent', `var(--${theme}-accent)`);
  
  // Update font families based on theme
  switch(theme) {
    case 'hodemei':
      root.style.setProperty('--font-header', 'Roboto, sans-serif');
      root.style.setProperty('--font-body', 'Noto Sans JP, sans-serif');
      break;
    case 'quxe':
      root.style.setProperty('--font-header', 'Crimson Text, serif');
      root.style.setProperty('--font-body', 'Lora, serif');
      break;
    case 'alsarejia':
      root.style.setProperty('--font-header', 'Space Mono, monospace');
      root.style.setProperty('--font-body', 'Work Sans, sans-serif');
      break;
    case 'laboratory':
      root.style.setProperty('--font-header', 'Fira Code, monospace');
      root.style.setProperty('--font-body', 'IBM Plex Sans, sans-serif');
      break;
  }
}
